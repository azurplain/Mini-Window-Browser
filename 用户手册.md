## 1. 安装与运行

### 环境要求

本软件基于 Microsoft WebView2 技术，您的电脑需要安装 **Microsoft Edge WebView2 Runtime**。

- Windows 10 / 11 系统通常已内置该组件。
- 如果软件无法打开并提示"无法创建 WebView2 环境"，请前往 [微软官网](https://developer.microsoft.com/zh-cn/microsoft-edge/webview2/) 下载安装 WebView2 Runtime。

### 文件说明

解压后，请确保文件夹内包含以下核心文件，且 **必须在同一目录下**：

| 文件名 | 说明 |
|---|---|
| `XiaoChuang.exe` | 主程序 |
| `WebView2Loader.dll` | WebView2 核心组件 |
| `config.ini` | 配置文件（首次运行后自动生成） |

### 如何运行

> ⚠️ **强烈建议**：右键点击 `XiaoChuang.exe`，选择 **"以管理员身份运行"**。

**原因**：全屏游戏（如《原神》、《英雄联盟》等）通常拥有较高的系统权限。如果小窗没有管理员权限，全局快捷键可能会被游戏拦截而失效。软件启动时会自动请求管理员权限，如果您拒绝了 UAC 提示，软件仍可运行，但热键功能在全屏游戏中将受限。

## 2. 界面布局介绍

软件窗口顶部为三层工具栏，总高度 70 像素：

```
┌─────────────────────────────────────────────────────┐
│  [拖拽条]                  [✥]                       │ ← 12px 拖拽区域
│  [标签页1] [标签页2] ...      [+] [-] [□] [×]       │ ← 28px 标签栏
│  [地址栏_____________] [☆][⌵][Go] [预设] [设置]     │ ← 30px 导航栏
├─────────────────────────────────────────────────────┤
│                                                     │
│                    网页内容区                         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### A. 拖拽条（最顶部 12px 白条）

- **移动手柄 [✥]**：位于白条正中央，默认隐藏。将鼠标悬停到白条中间区域即可显现，按住即可拖动窗口。
- 拖拽条其余区域也可用于拖动窗口（相当于标题栏）。

### B. 标签栏

- 显示当前打开的网页标签，每个标签右侧 20px 区域为关闭按钮（×）。
- **[+]** 新建标签页（默认打开主页）。
- **[-]** 最小化窗口到任务栏。
- **[□ / ❐]** 全屏切换按钮（详见第 6 节）。
- **[×]** 彻底关闭软件。

### C. 导航栏

- **地址栏**：输入网址后按回车或点击 [Go] 跳转。
  - 智能识别：有协议头直接访问，含点号自动补 `https://`，其余走必应搜索。
  - 单击地址栏自动全选文字，方便替换输入。
- **[☆ / ★]** 收藏/取消收藏当前网页。
- **[⌵]** 打开书签管理下拉列表。
- **[Go]** 导航按钮。
- **[预设]** 打开场景预设管理面板。
- **[设置]** 打开参数设置面板。

### D. 窗口缩放

- 窗口右下角 10×10 像素区域可拖拽调整窗口大小。

## 3. 核心功能：沉浸模式 (Immersion Mode)

这是本软件的灵魂功能，专为游戏挂机设计。

### 开启/关闭

按下快捷键 **`0`**（数字零）切换。

### 两种沉浸风格

在【设置】中可通过 "沉浸模式使用整体透明（替代挖孔）" 勾选框切换：

#### 风格一：挖孔穿透（默认）

- 窗口边框和工具栏全部隐藏，背景变透明。
- 屏幕上出现一个 **跟随鼠标的圆形区域**（"挖孔"），该区域显示网页内容。
- 鼠标点击圆孔 **以外** 的区域会直接 **穿透** 浏览器，作用于底层的游戏或桌面。
- 挖孔大小（半径）可在【设置】中调整，设为 0 则完全不显示内容（纯穿透）。

#### 风格二：整体透明（v1.3 完善）

- 整个窗口在鼠标移入时 **完全透明**（alpha=0），鼠标移出后恢复可见（alpha=255）。
- 适合需要大面积查看视频但不想被鼠标遮挡的场景。
- **v1.3 修复**：修复了退出沉浸模式后重新进入时，透明度状态可能不正确的问题。

### 沉浸模式注意事项

- 沉浸模式下 **无法** 切换全屏（需先退出沉浸模式）。
- 进入沉浸模式时窗口会自动补偿顶栏高度（向下平移 70px），退出时还原。
- 如果窗口已吸附屏幕顶部，进入沉浸模式时会自动贴顶。

## 4. 核心功能：场景预设 (Presets)

预设系统允许您保存当前的"环境状态"快照，一键在不同工作/娱乐场景间切换。

### 预设保存的内容

| 项目 | 说明 |
|---|---|
| 窗口位置与大小 | X、Y、宽、高 |
| 挖孔半径 | 沉浸模式的圆孔大小 |
| 吸附距离 | 边缘吸附阈值 |
| 自动暂停开关 | 老板键隐藏时是否暂停视频 |
| 沉浸风格 | 挖孔穿透 / 整体透明 |
| 全屏状态 | 是否处于全屏模式 |
| 热键禁用设置 | 输入时是否自动禁用热键 |
| 托盘模式 | 是否隐藏任务栏图标 |
| 默认主页 | 当前设定的主页 URL |
| 当前网址 | 应用预设时自动导航到此 URL |
| 全部热键绑定 | 7 个热键的修饰键+按键配置 |

### 如何使用

1. 点击工具栏 **[预设]** 按钮，打开管理窗口。
2. **新建**：先调整好窗口状态，点击 **[+ 新建预设]**，输入名称保存。
   - 系统会自动检测重名，防止误覆盖。
   - 全屏模式下新建预设，会保存全屏前的窗口位置。
3. **应用**：点击对应预设左侧的 **[➜]** 箭头，窗口瞬间恢复到保存时的状态。
4. **管理**：
   - 改名：直接编辑列表中的名称文本框，关闭窗口时自动保存。
   - 删除：点击右侧的 **[🗑]** 图标。
5. **还原默认**：点击底部的 **[↺ 还原默认状态]**，重置窗口为初始大小（1000×600）、位置（100,100）及所有默认设置。

> 💡 **提示**：预设不会保存书签数据。书签是全局共享的，所有预设使用同一份书签。

## 5. 书签与收藏系统 (Bookmarks)

### 一键收藏

点击导航栏的 **[☆]** 星星图标：

- **空心星 [☆]**：当前页未收藏，点击添加。
- **实心星 [★]**：已收藏，再次点击取消收藏。
- 切换标签页、跳转网页、应用预设时，星标状态自动同步。
- URL 匹配规则：去除末尾斜杠、忽略大小写后进行比对。

### 书签管理窗口

点击 **[⌵]** 按钮打开下拉列表：

- **[➜]** 跳转到对应网页。
- **编辑**：直接修改列表中的 URL 或标题文本框，修改即时保存。
- **[🗑]** 删除对应书签。
- 书签过多时，支持鼠标滚轮和滚动条上下滚动。
- 点击列表外任意区域自动关闭（失焦关闭）。

### 数据存储

所有书签保存在 `config.ini` 文件的 `[Bookmarks]` 段中。备份此文件即可保留全部书签。

## 6. 全屏模式

点击标签栏的 **[□]** 按钮（或当已全屏时显示为 **[❐]**）切换全屏：

- 全屏使用显示器的 **工作区**（rcWork），即排除任务栏后的可用区域。
- 沉浸模式下全屏按钮无效（需先退出沉浸模式）。
- 拖动窗口改变位置或大小后，全屏状态自动取消。
- 全屏前的窗口位置会被记录，退出全屏时自动还原。

## 7. 系统托盘模式 (v1.3 完善)

在【设置】中勾选 **"隐藏任务栏图标（最小化到托盘）"**：

- 任务栏上的窗口按钮会消失，取而代之的是系统托盘区域（通知栏）的小图标。
- **双击托盘图标**：切换窗口显示/隐藏（等同于老板键）。
- **右键托盘图标**：弹出菜单，可选择"显示/隐藏窗口"或"退出"。
- 预设中会保存托盘模式的开关状态。

## 8. 默认快捷键列表

> ⚠️ 当您在网页输入框或地址栏打字时（光标闪烁），全局热键会自动失效，防止误触。停止打字或点击网页空白处后自动恢复。此行为可在设置中通过"输入时自动禁用全局热键"开关控制。

### 全局控制

| 快捷键 | 功能 |
|---|---|
| `0`（数字零） | 开启/关闭 沉浸模式 |
| `9`（数字九） | 老板键（一键隐藏/显示窗口） |

### 视频控制（适配 Bilibili 等主流网站）

| 快捷键 | 功能 |
|---|---|
| `` ` ``（波浪号键） | 播放 / 暂停 |
| `5` | 后退 5 秒 |
| `6` | 快进 5 秒 |
| `7` | 上一集（B站合集适配） |
| `8` | 下一集（B站合集适配） |

### 老板键特殊行为

- 隐藏窗口时，如果开启了"老板键隐藏时自动暂停视频"（默认开启），会自动执行 JS 暂停视频。
- 窗口隐藏后，除老板键（显示/隐藏）外的所有其他热键自动注销，避免误触。

## 9. 自定义热键（含鼠标按键绑定）(v1.3 完善)

所有 7 个热键均可在【设置】中自定义：

1. 点击对应功能右侧的按钮，按钮显示"按下新键..."。
2. 按下您想要的 **键盘按键** 或 **鼠标按键**。
3. 按 **ESC** 可取消当前绑定操作，恢复原有热键。
4. 支持的组合修饰键：`Ctrl`、`Shift`、`Alt`（可任意组合）。

### 鼠标按键支持

| 鼠标按键 | 说明 |
|---|---|
| 鼠标中键（滚轮按下） | 显示为"鼠标中键" |
| 鼠标侧键1（前进键） | 显示为"鼠标侧键1" |
| 鼠标侧键2（后退键） | 显示为"鼠标侧键2" |

> 💡 鼠标按键热键通过低级鼠标钩子 (`WH_MOUSE_LL`) 实现全局捕获，即使在全屏游戏中也能正常工作（需管理员权限）。

## 11. 常见问题 (FAQ)

### Q1: 为什么按热键没反应？

**A:**
1. **检查输入状态**：在网页输入框或地址栏中打字时，热键会被自动屏蔽（可在设置中关闭此行为）。
2. **检查权限**：最常见原因是权限不足。请务必 **以管理员身份运行** 本软件，否则全屏游戏会拦截按键。
3. **检查输入法**：某些输入法可能拦截数字键。尝试切换到英文输入法。
4. **检查键位冲突**：如果其他软件也注册了相同的全局热键，可能导致冲突。在【设置】中更换为其他按键组合。

### Q2: 如何关闭标签页？

**A:** 点击标签页右侧的 **×** 区域即可关闭该标签。标签栏的 **[-]** 按钮功能为最小化窗口到任务栏。当只剩一个标签页时，关闭操作会重置该标签为默认主页，而不是退出程序。如需退出，请点击窗口最右侧的 **[×]** 按钮，或通过托盘图标右键菜单选择"退出"。

### Q3: 预设会保存我的书签吗？

**A:** 不会。预设 (Presets) 仅保存"窗口状态参数"和"热键配置"。书签 (Bookmarks) 是独立存储的全局数据，所有预设共享同一份书签。

### Q4: 视频快进/快退功能对所有网站有效吗？

**A:** 该功能通过注入 JavaScript 操作页面中的 `<video>` 元素实现：
- 对大多数标准 HTML5 播放器有效。
- 上一集/下一集功能针对 **Bilibili** 进行了特别优化（查找 `.bpx-player-ctrl-prev` / `.bpx-player-ctrl-next` 按钮）。
- 对于使用 Flash 或非标准播放器的网站可能无效。

### Q5: 如何备份我的数据？

**A:** 所有设置、书签和预设都保存在程序同目录下的 **`config.ini`** 文件中。备份此文件即可保留全部数据。该文件包含以下段落：
- `[Settings]`：基本设置（挖孔半径、吸附距离、主页等）
- `[Session]`：会话数据（标签页、窗口位置）
- `[Hotkeys]`：热键绑定
- `[Bookmarks]`：书签
- `[Presets]`：场景预设

### Q6: 吸附屏幕边缘后怎么拉开？

**A:** 按住 **Ctrl** 键拖动窗口即可临时脱离吸附。也可以在【设置】中将"吸附距离"调小或设为 0。

### Q7: 整体透明模式和挖孔模式有什么区别？

**A:**

| 特性 | 挖孔穿透（默认） | 整体透明 |
|---|---|---|
| 视觉效果 | 鼠标附近的圆形区域显示内容 | 鼠标在窗口上时完全隐藏，移开后完整显示 |
| 适用场景 | 只需局部查看视频画面 | 需要看完整画面，操作时隐藏 |
| 性能开销 | 较高（每帧更新 GDI 区域） | 较低（仅切换 alpha 值） |
| 切换方式 | 设置 → 勾选"沉浸模式使用整体透明" | 同左 |

### Q8: 为什么窗口总是显示在最顶层？

**A:** 这是"画中画"浏览器的核心设计。窗口创建时带有 `WS_EX_TOPMOST` 属性，确保它始终浮在其他窗口之上。这样您在玩游戏或使用其他软件时仍能看到小窗内容。

### Q9: 鼠标侧键绑定热键后，原来的浏览器前进/后退功能还在吗？

**A:** 如果将鼠标侧键绑定为小窗的热键，按下侧键时事件会被小窗拦截，不再传递给其他程序。如需恢复，请在【设置】中将对应热键改回键盘按键。

### Q10: 进入沉浸模式后怎么退出？

**A:** 按下沉浸模式热键（默认 `0`）即可退出。由于沉浸模式下鼠标操作会穿透窗口，此时只能通过热键退出。如果忘记了热键，可以使用老板键（默认 `9`）先隐藏再显示窗口，或通过托盘图标右键退出程序。

### Q11: 软件支持多显示器吗？

**A:** 支持。窗口吸附和全屏功能使用 `MonitorFromWindow` 自动检测当前所在显示器。弹窗（设置、预设、命名、书签）会自动居中于所属显示器的工作区，并确保不超出屏幕边界。
